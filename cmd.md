# Подсказки по командной строке

Команда смены директорий
```sh
cd c:\folder_name
```
Команда отображения текущей директории
```sh
pwd
```

Листинг текщей дириктории Windows:
```
dir
```
Linux, MacOs
```sh
ls
```

Удаление файла Windows:
```sh
del <filename>
```
в Linux, MacOs
```sh
rm <filename>
```
